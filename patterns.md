# 机器学习算法模式与设计模式 / ML Algorithm & Design Patterns

本文档提取了西瓜书、李航《机器学习方法》等五本教材中反复出现的具体技术、算法框架和设计思想。

---

## 梯度下降变体 / Gradient Descent Variants

### 批量梯度下降 (Batch GD)
**何时使用**: 小型数据集（能全部载入内存），需要稳定收敛
**如何运作**: 每次迭代使用全部训练样本计算真实梯度后更新参数
**关键公式**: $\theta_{t+1} = \theta_t - \eta \nabla_\theta J(\theta)$
**优缺点**: 收敛稳定但每步计算开销大，容易陷入局部鞍点
**来源**: 李航 附录A

### 随机梯度下降 (SGD)
**何时使用**: 大规模数据集，深度学习训练的默认选择
**如何运作**: 每次迭代随机抽样一个mini-batch计算梯度估计并更新参数
**关键公式**: $\theta_{t+1} = \theta_t - \eta g_t$，其中 $g_t$ 为mini-batch梯度
**优缺点**: 计算高效，梯度噪声有助于逃离局部最优点，但收敛路径震荡
**来源**: 李航 Ch29

### 动量法 (Momentum)
**何时使用**: 梯度方向频繁震荡的场景（如峡谷形损失面）
**如何运作**: 在SGD基础上累积历史梯度的指数加权移动平均作为更新方向
**关键公式**: $v_t = \beta v_{t-1} + (1-\beta)g_t$; $\theta_t = \theta_{t-1} - \eta v_t$
**优缺点**: 加速收敛，抑制震荡；需要额外调节动量系数 $\beta$
**来源**: 李航 Ch29

### Adam 优化器
**何时使用**: 深度学习训练的默认选择，适用于大多数网络结构
**如何运作**: 同时维护梯度的一阶矩（动量）和二阶矩（自适应学习率）估计
**关键公式**: 结合Momentum和RMSProp，$\beta_1=0.9, \beta_2=0.999, \varepsilon=10^{-8}$
**优缺点**: 收敛快、鲁棒、超参数不敏感；有时泛化性能不如调参良好的SGD+Momentum
**来源**: 李航 Ch29

## 交叉验证模式 / Cross-Validation Patterns

### K折交叉验证
**何时使用**: 数据量适中（数百到数万），需要可靠的模型评估
**如何运作**: 将数据均分为K份，依次以1份验证、K-1份训练，取K次结果的平均
**关键参数**: K=5或10（常用），K=N时为留一法
**优缺点**: 评估结果稳定，但计算开销是单次训练的K倍
**来源**: 西瓜书 Ch2; 李航 Ch1

### 分层K折 (Stratified K-Fold)
**何时使用**: 类别不平衡的分类问题
**如何运作**: 保证每折中各类别比例与原始数据集相同
**来源**: 西瓜书 Ch2

## 正则化模式 / Regularization Patterns

### L1正则化 (Lasso)
**何时使用**: 需要特征选择的场景，或已知大部分特征不相关
**如何运作**: 在损失函数中添加参数绝对值和惩罚: $J(\theta) + \lambda \|\theta\|_1$
**关键性质**: 产生稀疏解，等价于MAP估计中拉普拉斯先验
**来源**: 西瓜书 Ch11; 李航 Ch1

### L2正则化 (Ridge / 权重衰减)
**何时使用**: 防止过拟合的通用正则化，大多数模型的默认选择
**如何运作**: $J(\theta) + \frac{\lambda}{2} \|\theta\|_2^2$，等价于MAP估计中高斯先验
**关键性质**: 使参数值收缩但不为零，对应的优化问题仍是光滑凸函数
**来源**: 西瓜书 Ch3; 李航 Ch1

### Dropout (暂退法)
**何时使用**: 深层全连接网络的训练阶段
**如何运作**: 训练时以概率p随机将神经元输出置零；预测时所有权重乘以(1-p)
**关键原理**: 可视为指数数量子网络的隐式集成
**来源**: 李航 Ch23

### Early Stopping (早停法)
**何时使用**: 任何迭代训练的模型，深度学习中最常用
**如何运作**: 监控验证集误差，在连续N轮不降后停止，保存最优模型
**来源**: 李航 Ch23, Ch29

## 集成学习模式 / Ensemble Patterns

### Bagging (Bootstrap Aggregating)
**何时使用**: 高方差模型（如决策树），需降低过拟合风险
**如何运作**: 从训练集自助采样T个数据集，独立训练T个基学习器，投票/平均融合
**关键公式**: $f_{\text{bag}}(x) = \frac{1}{T}\sum_{t=1}^T f_t(x)$
**典型实现**: 随机森林（基学习器用决策树，且每次分裂仅随机考虑特征子集）
**来源**: 西瓜书 Ch8; 李航 Ch8

### Boosting (提升法)
**何时使用**: 高偏差弱学习器（如浅层决策树），需逐步提升性能
**如何运作**: 串行训练基学习器，每步根据前序模型的误差调整样本权重或残差
**关键变体**: AdaBoost（指数损失调整样本权重）、Gradient Boosting（拟合残差梯度）、XGBoost/LightGBM（工程优化版）
**来源**: 西瓜书 Ch8; 李航 Ch8

### Stacking (堆叠法)
**何时使用**: 多种异构基学习器的场景，希望用元学习器融合多源信息
**如何运作**: 第一层训练多个基学习器，第二层（元学习器）以基学习器输出为新特征进行训练
**注意**: 需使用交叉验证生成元学习器的训练特征以避免过拟合
**来源**: 西瓜书 Ch8

## 核技巧模式 / Kernel Trick

### 核方法通用框架
**何时使用**: 原始特征空间中线性不可分，但不希望显式计算高维映射
**如何运作**: 通过核函数 $K(x_i, x_j) = \langle \phi(x_i), \phi(x_j) \rangle$ 隐式地在高维空间做内积
**关键公式**: 原始问题中以 $\langle x_i, x_j \rangle$ 出现的位置，直接替换为 $K(x_i, x_j)$
**常用核函数**:
- 线性核: $K(x, y) = x^{\mathrm{T}}y$
- 多项式核: $K(x, y) = (x^{\mathrm{T}}y + c)^d$
- RBF/高斯核: $K(x, y) = \exp(-\gamma \|x - y\|^2)$
- Sigmoid核: $K(x, y) = \tanh(\alpha x^{\mathrm{T}}y + c)$
**优缺点**: 无需知道映射函数 $\phi$ 的显式形式，但计算复杂度为 $O(N^2)$（核矩阵）
**来源**: 西瓜书 Ch6; 李航 Ch7

### 表示定理 (Representer Theorem)
**何时使用**: 核方法 + L2正则化的优化问题
**如何运作**: 正则化风险最小化问题的最优解必为核函数在训练点上的线性组合: $f(x) = \sum_{i=1}^N \alpha_i K(x, x_i)$
**来源**: 西瓜书 Ch6

## 降维模式 / Dimensionality Reduction Patterns

### PCA范式
**何时使用**: 特征间高度相关，需去除冗余或数据可视化
**如何运作**: 对中心化数据的协方差矩阵做特征值分解，取前k大特征值对应的特征向量作为投影方向
**关键公式**: $Y = V_k^{\mathrm{T}}X$，信息保留比: $\frac{\sum_{i=1}^k \lambda_i}{\sum_{i=1}^p \lambda_i}$
**来源**: 西瓜书 Ch10; 李航 Ch16

### SVD范式
**何时使用**: 任意形状矩阵的低秩近似、推荐系统、话题分析
**如何运作**: 将矩阵分解为 $A \approx U_k \Sigma_k V_k^{\mathrm{T}}$，利用Eckart-Young定理保证在Frobenius范数下最优
**来源**: 李航 Ch15, Ch21

### 非线性降维选择指南
- **核PCA**: 数据在低维非线性流形上，但目标仍是全局方差最大
- **ISOMAP**: 希望保留测地距离（适合swiss roll等卷曲结构）
- **t-SNE**: 数据可视化为主，保局部结构
**来源**: 西瓜书 Ch10

## 聚类模式 / Clustering Patterns

### K-Means范式 (基于中心的聚类)
**何时使用**: 簇形状近似球形、各簇大小相近、已知簇数K
**如何运作**: 交替执行"分配样本到最近中心"和"更新中心为簇内均值"直至收敛
**关键公式**: $\min_C \sum_{k=1}^K \sum_{x \in C_k} \|x - \mu_k\|^2$
**优缺点**: 简单高效 $O(NKd)$，但对初始化和离群点敏感，只能发现凸形簇
**来源**: 西瓜书 Ch9; 李航 Ch14

### 层次聚类范式
**何时使用**: 不确定簇数，希望看到数据的层次结构
**如何运作** (凝聚法): 初始每个样本各成一簇，每次合并距离最近的两簇
**簇间距离**: 单链接 (min)、全链接 (max)、平均链接、Ward法
**来源**: 西瓜书 Ch9; 李航 Ch14

### 高斯混合模型 + EM范式 (软聚类)
**何时使用**: 需要软分配（不确定性量化）而非硬分配
**如何运作**: E步计算每个样本属于每个高斯成分的后验概率，M步更新各成分的均值、方差和混合系数
**来源**: 西瓜书 Ch9; 李航 Ch9

## 贝叶斯推断模式 / Bayesian Inference Pattern

### 朴素贝叶斯范式
**何时使用**: 特征维度高但特征间近似独立（如文本分类）
**如何运作**: 利用特征条件独立假设，$P(Y|X) \propto P(Y) \prod_i P(X_i|Y)$；使用MAP决策
**关键技巧**: 拉普拉斯平滑防止零概率——$P(X_i|Y) = \frac{N_{i,Y} + \alpha}{N_Y + \alpha \cdot |V|}$
**来源**: 西瓜书 Ch7; 李航 Ch4

### 贝叶斯推断通用框架
**何时使用**: 需要量化预测不确定性，或数据量少需要先验引导
**如何运作**: 先验 $P(\theta)$ + 似然 $P(D|\theta)$ → 后验 $P(\theta|D) \propto P(D|\theta)P(\theta)$
**共轭先验技巧**: 选择与似然共轭的先验分布族，使后验保持同族解析形式
**来源**: 西瓜书 Ch7; 李航 Ch4

## EM模式 (Expectation-Maximization Pattern)

**何时使用**: 模型含隐变量或缺失数据，直接对观测数据的似然优化困难
**如何运作**:
1. **E步**: 基于当前参数估计隐变量的后验分布（期望充分统计量）
2. **M步**: 最大化完整数据的期望对数似然，更新参数
3. 交替执行直至收敛
**关键性质**: 每步迭代保证似然单调不减，但收敛到局部最优
**典型应用**: GMM参数估计、HMM的Baum-Welch算法、缺失数据填充
**来源**: 西瓜书 Ch7; 李航 Ch9

---

## 反向传播模式 (Backpropagation Pattern)

**何时使用**: 任何基于梯度的神经网络训练
**如何运作**:
1. **正向传播**: 从输入到输出逐层计算激活值
2. **计算损失**: 比较输出与目标的差异
3. **反向传播**: 利用链式法则从输出层到输入层逐层计算梯度
4. **参数更新**: 使用优化器（SGD/Adam）更新所有参数
**实现技巧**: 在计算图上，正向和反向传播均为张量沿有向边的流动
**梯度消失处理**: ReLU激活 + LSTM门控 + 残差连接 + 批量归一化
**来源**: 李航 Ch23, Ch29

## 注意力机制模式 / Attention Mechanism Pattern

**何时使用**: 序列到序列映射、需要动态聚焦输入特定位置的任何任务
**如何运作**: 对编码器每个位置的隐状态 $h_i$ 计算与解码器当前状态 $s_t$ 的相关度分数，经softmax归一化后作为权重对 $h_i$ 做加权求和
**关键公式**: $c_t = \sum_i \alpha_{ti} h_i$, $\alpha_{ti} = \frac{\exp(\text{score}(s_t, h_i))}{\sum_j \exp(\text{score}(s_t, h_j))}$
**分数函数**:
- 点积 (dot-product): $s^{\mathrm{T}}h$
- 缩放点积 (scaled): $s^{\mathrm{T}}h / \sqrt{d}$
- 加性 (additive/Bahdanau): $v^{\mathrm{T}}\tanh(W[s; h])$
**自注意力**: $Q, K, V$ 均来自同一序列，$c = \text{softmax}(QK^{\mathrm{T}}/\sqrt{d_k})V$
**多头**: 用不同的投影矩阵并行做h次自注意力，拼接结果——允许模型同时关注不同表示子空间的信息
**来源**: 李航 Ch26

## 预训练-微调模式 / Pretrain-Finetune Pattern

**何时使用**: 下游标注数据稀缺但有大量无标注数据的场景（如NLP、CV）
**如何运作**:
1. **预训练**: 在大规模无标注语料上用自监督任务（MLM、NSP、对比学习等）训练模型
2. **微调**: 在下游任务的少量标注数据上继续训练，通常配合较小的学习率
**典型模型**: BERT（MLM预训练→分类/QA微调）、GPT（自回归语言模型预训练→生成/理解微调）
**关键优势**: 利用通用语言知识降低对标注数据的需求
**来源**: 李航 Ch27

---

## 对抗训练模式 / Adversarial Training Pattern

**何时使用**: 数据生成（图像、文本）、域适应、提高模型稳健性
**如何运作**: 两个网络交替更新——生成器试图最小化 $\log(1-D(G(z)))$，判别器试图最大化 $\log D(x) + \log(1-D(G(z)))$
**关键技巧**: 
- 判别器更新S次后生成器更新1次
- 生成器的目标替换为最大化 $\log D(G(z))$（缓解早期梯度消失）
- 使用批量归一化和LeakyReLU
**训练诊断**: 若判别器loss接近0则过强，若生成器loss剧烈震荡则不稳定
**来源**: 李航 Ch28

## 模型选择与超参数调优 / Model Selection & HP Tuning

### 奥卡姆剃刀原则
**何时使用**: 多个性能相近的模型之间做选择
**原则**: 在性能相当的模型中选最简单的——简单模型泛化能力通常更好
**来源**: 西瓜书 Ch2

### 网格搜索 vs 随机搜索
**何时使用**: 超参数空间探索
- **网格搜索**: 超参数维度低（≤3），每维候选值少
- **随机搜索**: 超参数维度高，部分参数重要性远大于其他
**来源**: 西瓜书 Ch2
