# 机器学习速查表 / Machine Learning Cheatsheet

## 1. 决策规则 / Decision Rules

| 场景 | 决策 | 原因 |
|------|------|------|
| 数据线性可分 | 先用感知机/逻辑斯谛回归/SVM（线性核） | 简单模型更易解释且不易过拟合 |
| 数据非线性但特征维度不高 | SVM + RBF核 | 核技巧高效处理非线性，低维时计算可接受 |
| 特征维度极高且稀疏（如文本） | 线性模型（逻辑斯谛回归、线性SVM） | 高维空间已近似线性可分 |
| 样本少（<1000）但标注可靠 | 朴素贝叶斯 / SVM / 决策树 | 线性模型和简单树在小样本下更稳健 |
| 样本多（>100万） | 深度学习（CNN/Transformer） | 大样本是深度学习的燃料，表示学习优势显著 |
| 需要概率输出 | 逻辑斯谛回归（二分类）/ softmax回归（多分类） | 原生输出校准概率 |
| 需要特征重要性解释 | 决策树 / 随机森林 / L1正则化线性模型 | 树模型天然提供特征重要性，L1产生稀疏解 |
| 存在大量缺失值 | XGBoost / LightGBM | 树模型原生处理缺失值 |
| 图像数据 | CNN（ResNet/VGG） | 卷积的归纳偏置（局部性、平移等变性）适合图像 |
| 序列/文本数据 | Transformer / LSTM | 自注意力捕获长距离依赖，预训练模型迁移学习 |
| 类别严重不平衡 | 重采样（SMOTE）+ Focal Loss + 调整类别权重 | 避免模型偏向多数类 |
| 数据量小+需要生成新数据 | GAN / VAE / 数据增强 | 生成模型学习数据分布用于采样 |

## 2. 算法选择指南 / Algorithm Selection Guide

### 分类问题

| 算法 | 适用场景 | 核心优势 | 核心局限 | 来源 |
|------|---------|---------|---------|------|
| 逻辑斯谛回归 | 二/多类分类基线 | 简单、概率输出、可解释 | 线性假设 | 西瓜书Ch3; 李航Ch6 |
| 朴素贝叶斯 | 文本分类、高维稀疏数据 | 训练极快、小样本有效 | 特征独立假设 | 西瓜书Ch7; 李航Ch4 |
| KNN | 小数据、低维、非参数 | 无需训练、天然多类 | 对维度和噪声敏感 | 西瓜书Ch10; 李航Ch3 |
| SVM | 中小规模数据、高维 | 最大间隔理论保证、核技巧 | 大规模数据慢 | 西瓜书Ch6; 李航Ch7 |
| 决策树 | 需要可解释性 | 可视化、处理混合特征 | 易过拟合 | 西瓜书Ch4; 李航Ch5 |
| 随机森林 | 通用分类/回归 | 高精度、抗过拟合、特征重要性 | 模型大、推理慢 | 西瓜书Ch8 |
| XGBoost/LightGBM | 结构化数据竞赛首选 | 极高性能、原生缺失值处理 | 超参数多需调参 | 西瓜书Ch8; 李航Ch8 |
| 多层感知器 | 非线性但非图像/序列 | 通用近似能力 | 需大量调参 | 西瓜书Ch5; 李航Ch23 |
| CNN | 图像分类/识别 | 参数效率极高 | 仅适合网格结构 | 李航Ch24 |
| RNN/LSTM/Transformer | 文本分类/序列标注 | 建模时序依赖 | 训练较慢 | 李航Ch25-26 |

### 回归问题

| 算法 | 核心公式 | 适用场景 |
|------|---------|---------|
| 线性回归 | $\hat{y} = w^{\mathrm{T}}x + b$ | 基线模型 |
| Ridge回归 | $\min \|Xw - y\|^2 + \lambda\|w\|_2^2$ | 多重共线性数据 |
| Lasso回归 | $\min \|Xw - y\|^2 + \lambda\|w\|_1$ | 自动特征选择 |
| SVR | 核方法 + $\epsilon$-insensitive loss | 非线性小数据 |
| 树模型 | 随机森林回归 / GBDT | 结构化数据首选 |

### 聚类问题

| 算法 | 适用场景 | 核心参数 |
|------|---------|---------|
| K-Means | 球形簇、预知簇数 | K（簇数） |
| DBSCAN | 任意形状簇、含噪声 | $\epsilon$（邻域半径）, minPts |
| 层次聚类 | 需要层次结构 | 距离度量、链接方式 |
| GMM (EM) | 需要软分配+概率密度 | K（成分数） |

### 降维问题

| 算法 | 类型 | 适用场景 |
|------|------|---------|
| PCA | 线性 | 去除线性冗余、基线降维 |
| SVD | 线性 | 稀疏矩阵、推荐系统、话题分析 |
| 核PCA | 非线性 | 非线性流形+全局方差 |
| t-SNE | 非线性 | 高维数据可视化 |
| LSA (截断SVD) | 线性 | 文本话题分析 |
| NMF | 线性（非负） | 话题解释性强 |

## 3. 关键公式速查 / Key Formulas at a Glance

### 损失函数

| 名称 | 公式 | 使用场景 |
|------|------|---------|
| 0-1损失 | $L = \mathbb{I}[y \neq \hat{y}]$ | 理论分析基准 |
| 平方损失 | $L = (y - \hat{y})^2$ | 线性回归 |
| 交叉熵损失 | $L = -\sum_k y_k \log \hat{y}_k$ | 分类（配合softmax） |
| 合页损失 | $L = \max(0, 1 - yf(x))$ | SVM |
| 指数损失 | $L = \exp(-yf(x))$ | AdaBoost |
| 逻辑斯谛损失 | $L = \log(1 + \exp(-yf(x)))$ | 逻辑斯谛回归 |

### 核心算法公式

| 算法 | 核心公式 |
|------|---------|
| 线性回归闭式解 | $\hat{w} = (X^{\mathrm{T}}X)^{-1}X^{\mathrm{T}}y$ |
| 逻辑斯谛回归 | $P(Y=1\|x) = \frac{1}{1+e^{-w^{\mathrm{T}}x}}$ |
| 朴素贝叶斯 | $P(Y\|X) \propto P(Y)\prod_i P(X_i\|Y)$ |
| SVM原问题 | $\min_{w,b} \frac{1}{2}\|w\|^2, \text{ s.t. } y_i(w^{\mathrm{T}}x_i + b) \geq 1$ |
| SVM对偶 | $\max_\alpha \sum_i \alpha_i - \frac{1}{2}\sum_{i,j}\alpha_i\alpha_j y_i y_j \langle x_i, x_j\rangle, \text{ s.t. } \alpha_i \geq 0$ |
| KKT条件 | $\alpha_i(y_i(w^{\mathrm{T}}x_i + b) - 1) = 0$（仅支持向量 $\alpha_i > 0$） |
| 核函数RBF | $K(x, y) = \exp(-\gamma\|x - y\|^2)$ |
| EM算法 | E步: $Q(\theta\|\theta^{(t)}) = E_{Z\|X,\theta^{(t)}}[\log P(X,Z\|\theta)]$; M步: $\theta^{(t+1)} = \arg\max Q$ |
| PCA投影 | $Y = V_k^{\mathrm{T}}X$, $V_k$ = 协方差矩阵前k大特征向量 |
| SVD截断 | $A \approx U_k \Sigma_k V_k^{\mathrm{T}}$（Frobenius范数下最优） |
| PageRank | $R = d M^{\mathrm{T}}R + \frac{1-d}{N}\mathbf{1}$ |
| 注意力 | $c_t = \sum_i \mathrm{softmax}(\frac{q^{\mathrm{T}}k_i}{\sqrt{d_k}}) v_i$ |
| LSTM | $c_t = f_t \odot c_{t-1} + i_t \odot \tilde{c}_t$, $h_t = o_t \odot \tanh(c_t)$ |

### 评估指标

| 指标 | 公式 | 说明 |
|------|------|------|
| 准确率 | $\frac{TP+TN}{TP+TN+FP+FN}$ | 平衡数据集可用 |
| 精确率 | $\frac{TP}{TP+FP}$ | 关注误报代价 |
| 召回率 | $\frac{TP}{TP+FN}$ | 关注漏报代价 |
| F1 | $\frac{2PR}{P+R}$ | 精确率与召回率调和 |
| AUC | ROC曲线下面积 | 整体排序质量 |

### 泛化理论

| 概念 | 关键不等式 |
|------|-----------|
| 泛化误差上界 | $R(f) \leq \hat{R}(f) + \sqrt{\frac{d(\log(2N/d)+1) - \log(\eta/4)}{N}}$ |
| 偏差-方差分解 | $\mathbb{E}[(y - \hat{f})^2] = \text{Bias}^2 + \text{Var} + \sigma^2_\epsilon$ |

## 5. 常见故障诊断 / Common Pitfalls & Diagnostics

### 训练不收敛 / 收敛过慢
- **检查**: 学习率是否过大（loss震荡）或过小（loss下降极慢）
- **行动**: 从 $10^{-3}$ 开始，对数尺度调整；使用学习率调度器（余弦退火、阶梯衰减）
- **检查**: 梯度是否消失/爆炸——监控梯度范数
- **行动**: 使用梯度裁剪（clipnorm）；换用ReLU；加Batch Normalization/残差连接

### 过拟合 (训练误差低、验证误差高)
- **检查**: 训练集太小或模型太复杂
- **行动**: 增加数据（增强）；添加正则化（Dropout/L2/早停）；降低模型容量
- **经验法则**: 若训练样本数 < 参数量的10倍，需要更强的正则化

### 欠拟合 (训练误差和验证误差都高)
- **检查**: 模型容量不足或特征不够
- **行动**: 增加模型复杂度（更多层/神经元）；构造更多特征（多项式/交叉特征）；减少正则化

### 数据不平衡
- **检查**: 各类样本比例——若少数类占比 < 10%
- **行动**: 重采样（SMOTE）；使用加权损失/类别权重；使用AUC而不是Accuracy评估

### 深度学习特有
- **模式崩塌 (GAN)**: 生成器只生成少数模式的样本 → 使用WGAN/WGAN-GP
- **RNN不捕获长距离依赖**: → 换用LSTM/GRU，或直接用Transformer
- **验证loss稳定但训练loss仍在降**: → 过拟合，增强正则化
- **训练集上准确率高但测试集低且差异大**: → 检查训练/测试分布是否一致，数据增强是否合理

### 数据预处理速查
| 数据类型 | 预处理 |
|---------|--------|
| 连续数值特征 | 标准化 (mean=0, std=1) 或 归一化 (min=0, max=1) |
| 类别特征 | One-hot编码（类别数少）或 Embedding（类别数多） |
| 文本 | 分词 → TF-IDF 或 词向量/预训练模型编码 |
| 图像 | 缩放至固定尺寸 → 像素值归一化 (/255) → 标准化 |
| 缺失值 | 树模型：原生处理；其他：均值/中位数填充 或 指示变量 |
| 时序 | 滑动窗口构造样本 → 平稳化（差分）→ 归一化 |
