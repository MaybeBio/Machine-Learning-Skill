# 第9章: 贝叶斯分类器 / Bayesian Classifiers

> **融合来源**: 西瓜书第7章(贝叶斯分类器) + 南瓜书第7章 + 李航第4章(朴素贝叶斯法)

## 一、入门理解 — 周志华《机器学习》(西瓜书)

### 1.1 贝叶斯决策论

贝叶斯分类器基于概率框架：由**贝叶斯定理**从先验概率 $P(c)$ 和类条件概率 $P(\mathbf{x}|c)$ 计算后验概率 $P(c|\mathbf{x})$：

$$P(c|\mathbf{x}) = \frac{P(c) P(\mathbf{x}|c)}{P(\mathbf{x})}$$

其中 $P(\mathbf{x}) = \sum_c P(c)P(\mathbf{x}|c)$ 为证据因子，与类别标记无关。

- **最小错误率准则**：$\hat{c} = \arg\max_c P(c|\mathbf{x})$
- **最小风险准则**（考虑不同误判的代价）：$R(c_i|\mathbf{x}) = \sum_j \lambda_{ij} P(c_j|\mathbf{x})$，选择 $\hat{c} = \arg\min_c R(c|\mathbf{x})$。其中 $\lambda_{ij}$ 是将真实类 $c_j$ 误判为 $c_i$ 的代价。

西瓜书指出：贝叶斯最优分类器 $h^*(\mathbf{x}) = \arg\max_c P(c|\mathbf{x})$ 的泛化误差称为**贝叶斯误差**，是任何分类器所能达到的最优下界。

### 1.2 朴素贝叶斯

估计 $P(\mathbf{x}|c)$ 的困难：$\mathbf{x}$ 是所有属性的联合取值，组合数随属性数指数增长。**朴素贝叶斯**用一个"天真"的假设避开此困难——**属性条件独立假设**：每个属性独立地对分类结果产生影响。

$$P(c|\mathbf{x}) \propto P(c) \prod_{i=1}^{d} P(x_i|c)$$

先验概率 $P(c) = \frac{|D_c| + \lambda}{|D| + \lambda N}$（拉普拉斯平滑修正，$\lambda \geq 0$）。$\lambda=1$ 时即经典的拉普拉斯平滑(Laplace smoothing)，$\lambda<1$ 为Lidstone平滑。

朴素贝叶斯的判定：$\hat{c} = \arg\max_c P(c) \prod_{i=1}^{d} P(x_i|c)$

### 1.3 半朴素贝叶斯分类器

适当放松独立性假设，考虑部分属性间的依赖关系：
- **SPODE**(Super-Parent ODE)：所有属性都依赖于一个"超父"属性
- **TAN**(Tree-Augmented Naive Bayes)：基于条件互信息构建最大带权生成树
- **AODE**(Averaged One-Dependent Estimator)：SPODE的集成版本

### 1.4 贝叶斯网(Bayesian Network)

用**有向无环图**(DAG)编码属性间的依赖关系。结点→属性，有向边→依赖关系。每个结点有一个**条件概率表**(CPT)，记录该结点给定父结点组合下的条件概率。

贝叶斯网的联合概率分解：$P(x_1, \ldots, x_d) = \prod_{i=1}^{d} P(x_i | \pi_i)$，其中 $\pi_i$ 是 $x_i$ 的父结点集合。

**三个基本问题**：
- **结构学习**：学出图结构（评分搜索：BIC/MDL评分+爬山/贪心搜索）
- **参数学习**：给定结构估计CPT（MLE/Bayesian estimation）
- **推断**：计算后验概率（精确推断：变量消除；近似推断：MCMC/Gibbs采样）

西瓜书用d-分离判断条件独立性：若路径被阻塞(阻塞结点=Head-to-Head未给定或Tail-to-Tail/Head-to-Tail已给定)，则两变量关于给定集条件独立。

## 二、公式推导 — 南瓜书补充

### 2.1 朴素贝叶斯的MLE推导

南瓜书推导：对于离散属性，$P(x_i|c) = \frac{|D_{c,x_i}|}{|D_c|}$（MLE，无平滑）。加上Dirichlet先验得到贝叶斯估计：$P(x_i|c) = \frac{|D_{c,x_i}| + \lambda}{|D_c| + \lambda N_i}$，其中 $N_i$ 为属性 $i$ 的取值数。这等价于MAP估计。

### 2.2 贝叶斯决策论的完整推导

南瓜书补充：在0-1损失下，条件风险 $R(c|\mathbf{x}) = 1 - P(c|\mathbf{x})$，最小化条件风险 = 最大化后验概率 → 最小错误率准则。

在一般损失函数下：$R(c_i|\mathbf{x}) = \sum_{j=1}^{N} \lambda_{ij} P(c_j|\mathbf{x})$

**判别式模型 vs 生成式模型的形式化对比**：
- 生成式：$P(c|\mathbf{x}) = \frac{P(\mathbf{x}|c)P(c)}{P(\mathbf{x})}$ —— 对$P(\mathbf{x}|c)$建模
- 判别式：直接建模 $P(c|\mathbf{x})$ 或 $f(\mathbf{x}) = \text{sign}(g(\mathbf{x}))$

### 2.3 AODE的预测公式

南瓜书推导AODE：$\hat{c} = \arg\max_c \sum_{i=1}^{d} P(c, x_i) \prod_{j=1}^{d} P(x_j | c, x_i)$，其中 $P(c, x_i) = \frac{|D_{c, x_i}| + 1}{|D| + N \times N_i}$，$P(x_j|c, x_i) = \frac{|D_{c, x_i, x_j}| + 1}{|D_{c, x_i}| + N_j}$

## 三、理论深化 — 李航《机器学习方法》

### 3.1 朴素贝叶斯的统计学习三要素（李航Ch4）

**模型**：条件概率分布 $P(Y=c_k|X=\mathbf{x}) = \frac{P(Y=c_k) \prod_j P(X^{(j)}=x^{(j)} | Y=c_k)}{\sum_k P(Y=c_k) \prod_j P(X^{(j)}=x^{(j)} | Y=c_k)}$

**策略**：极大似然估计（MLE）或贝叶斯估计（加入Dirichlet先验 → 拉普拉斯平滑）。李航指出贝叶斯估计下的条件概率估计为 $P_\lambda(X^{(j)}=a_{jl}|Y=c_k) = \frac{\sum_i \mathbb{I}(x_i^{(j)}=a_{jl}, y_i=c_k) + \lambda}{\sum_i \mathbb{I}(y_i=c_k) + S_j\lambda}$，$S_j$为属性$j$的取值数。

**算法**：计算先验 $P(Y=c_k)$ 和条件概率 $P(X^{(j)}|Y=c_k)$，对输入 $\mathbf{x}$ 计算各类后验概率的分子（分母相同可忽略），取最大。

### 3.2 李航对条件独立性假设的讨论

李航指出：条件独立性假设虽然"天真"，但在实践中朴素贝叶斯通常表现良好，尤其在文本分类(如垃圾邮件过滤)中是经典方法。原因：（1）分类决策仅需比较后验概率的相对大小，即使概率估计不准，只要大小顺序正确即可；（2）属性的依赖关系可能相互抵消。

### 3.3 贝叶斯方法的统一视角（PPT补充）

生成式方法的训练 = 联合概率 $P(\mathbf{x}, y)$ 的密度估计。这比判别式方法（仅需决策边界）更难（需要更多数据），但优势在于可以检测异常样本、处理缺失数据、生成新样本。

## 四、综合总结

### 算法对比

| 方法 | 独立性假设 | 模型复杂度 | 适用场景 |
|------|-----------|-----------|---------|
| 朴素贝叶斯 | 完全独立 | 极低 | 文本分类、高维稀疏数据 |
| SPODE | 依赖一个超父 | 低 | 有明确关键属性时 |
| TAN | 树形依赖 | 中 | 一般分类任务 |
| AODE | SPODE集成 | 中 | 稳定提升NB性能 |
| 贝叶斯网 | 任意DAG | 高 | 需要理解因果关系时 |
| 逻辑回归 | 无独立性假设 | 中 | 概率校准良好 |

### 关键结论

1. **贝叶斯最优分类器是理论上界**，贝叶斯误差不可逾越
2. **朴素贝叶斯的"天真"假设在实践中往往足够好**，尤其对小数据和文本分类
3. **拉普拉斯平滑不可或缺**——避免零概率导致乘积为零
4. **贝叶斯网可编码任意依赖结构**，但结构学习和精确推断在计算上都是NP-hard
5. **生成式 vs 判别式**：数据少时生成式可能更好（利用了模型假设），数据多时判别式通常更优
6. **概率输出是核心优势**：可带置信度的预测
