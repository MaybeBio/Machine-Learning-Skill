> **融合来源**: 李航第11章(条件随机场) + 清华大学PPT

## 一、入门理解 (无西瓜书覆盖，自行引入)

### 1.1 从HMM到CRF

HMM是**生成模型**：建模$P(O,I)$，依赖很强的观测独立性假设。这在实际中常常不成立——一个词(观测)的标签(状态)不仅取决于词本身，还依赖于上下文。

**CRF是判别模型**：直接建模$P(I|O)$——给定观测序列，预测状态序列。CRF通过**特征函数**灵活编码观测和状态间的任意依赖关系，"有什么特征就写什么特征"，不需要假设它们的独立性。

### 1.2 概率无向图模型(MRF)基础

CRF定义在无向图上。无向图的三种马尔可夫性：

- **成对马尔可夫性**：不相邻结点在给定其他结点下条件独立
- **局部马尔可夫性**：结点仅被其邻居"屏蔽"
- **全局马尔可夫性**：被分离集隔开的结点集条件独立

**Hammersley-Clifford定理**：无向图的联合概率分布可因子分解为最大团(clique)上的势函数乘积：$P(Y) = \frac{1}{Z} \prod_C \Psi_C(Y_C)$，其中$Z = \sum_Y \prod_C \Psi_C(Y_C)$是归一化因子。

### 1.3 线性链CRF

设观测序列$X=(x_1,\ldots,x_T)$，状态序列$Y=(y_1,\ldots,y_T)$。线性链CRF的条件概率：

$$P(Y|X) = \frac{1}{Z(X)} \exp\left(\sum_{t=1}^{T} \sum_{k=1}^{K} w_k f_k(y_{t-1}, y_t, X, t)\right)$$

其中$f_k$是特征函数（可编码任意$y_{t-1}$与$y_t$的转移模式、$y_t$与$X$任意位置的关系），$w_k$是其权重。

## 二、公式推导

### 2.1 线性链CRF的参数化形式

化简特征函数：$P(Y|X) = \frac{1}{Z(X)} \exp\left(\sum_{t} \sum_k \lambda_k t_k(y_{t-1},y_t,X,t) + \sum_t \sum_l \mu_l s_l(y_t,X,t)\right)$

其中$t_k$为**转移特征**(与$y_{t-1},y_t$相关)，$s_l$为**状态特征**(仅与$y_t$相关)。统一写作：

$$P(Y|X) = \frac{1}{Z(X)} \exp\left(\sum_{k=1}^{K} w_k F_k(Y,X)\right)$$

其中$F_k(Y,X) = \sum_{t=1}^{T} f_k(y_{t-1},y_t,X,t)$。

### 2.2 CRF的矩阵形式（简化形式）

为便于计算，将CRF表示为矩阵。对每个位置$t$定义矩阵$\mathbf{M}_t$：

$$M_t(y_{t-1}, y_t|X) = \exp\left(\sum_k w_k f_k(y_{t-1}, y_t, X, t)\right)$$

则：$P(Y|X) = \frac{1}{Z(X)} \prod_{t=1}^{T} M_t(y_{t-1}, y_t|X)$，且 $Z(X) = [\mathbf{M}_1 \mathbf{M}_2 \cdots \mathbf{M}_T]_{\text{start,stop}}$

### 2.3 CRF的前向-后向算法

前向向量$\boldsymbol{\alpha}_t$：$\alpha_t(y_t) = \sum_{y_{t-1}} \alpha_{t-1}(y_{t-1}) M_t(y_{t-1}, y_t|X)$

后向向量$\boldsymbol{\beta}_t$：$\beta_t(y_t) = \sum_{y_{t+1}} M_{t+1}(y_t, y_{t+1}|X) \beta_{t+1}(y_{t+1})$

$$P(Y|X) = \frac{1}{Z(X)} \exp\left(\sum_t \sum_k w_k f_k\right), \quad Z(X) = \boldsymbol{\alpha}_T \cdot \mathbf{1}$$

### 2.4 CRF的参数学习

**IIS**(改进迭代尺度法)：与最大熵模型的参数估计相似，每次更新$\delta_k = \frac{1}{M}\log\frac{\tilde{E}[f_k]}{E[f_k]}$

**拟牛顿法(BFGS)**：直接优化对数似然，收敛更快。对数似然：

$$\ell(\mathbf{w}) = \sum_{Y} \tilde{P}(Y,X) \sum_k w_k F_k(Y,X) - \log Z(X)$$

梯度：$\frac{\partial \ell}{\partial w_k} = \tilde{E}[F_k] - E_{P_w}[F_k]$（经验期望 - 模型期望）

### 2.5 CRF的预测：Viterbi解码

$\delta_t(y_t) = \max_{y_1,\ldots,y_{t-1}} \sum_{t'=1}^{t} \sum_k w_k f_k(y_{t'-1}, y_{t'}, X, t')$

递推式类似于HMM的Viterbi，只是用特征函数权重代替了转移/观测概率。

## 三、理论深化

### 3.1 CRF vs HMM 深度对比

| 维度 | HMM | CRF |
|------|-----|-----|
| 模型类型 | 生成式($P(X,Y)$) | 判别式($P(Y|X)$) |
| 独立性假设 | 强(观测独立、一阶马尔可夫) | 弱(特征函数任意) |
| 特征设计 | 固定(转移+观测矩阵) | 灵活(任意特征函数) |
| 全局最优 | EM→局部最优 | 似然函数凸→**全局最优** |
| 数据需求 | 少(模型假设强) | 多(特征灵活需数据支撑) |

### 3.2 标记偏置问题

PPT补充：MEMM(最大熵马尔可夫模型)存在**标记偏置问题**(label bias problem)——由于每步做局部归一化，倾向于选择后续转移少的状态。CRF通过做**全局归一化**($Z(X)$在整条序列上归一化)彻底解决此问题。

### 3.3 CRF的优越性

- 对所有特征做全局归一化 → 无标记偏置
- 对数似然函数是凸函数 → 全局最优解
- 特征设计灵活 → 可以编码任意领域知识
- 缺点：训练计算量大（需要前向-后向计算$Z(X)$和梯度）

---

## 四、综合总结

### 关键结论

1. **CRF = 无向图 + 判别式 + 全局归一化**，是序列标注的最经典模型
2. **特征函数给CRF极大的灵活性**，且似然函数保持凸性→全局最优
3. **CRF的计算核心仍为前向-后向**（动态规划），只是在特征权重的指数空间中
4. **Viterbi解码与HMM一致**（本质上都是DP/最短路径）
5. **HMM → MEMM → CRF**：从生成式到判别式的进化，从局部归一化到全局归一化
6. **现代BiLSTM-CRF、BERT-CRF**：深度特征提取+CRF序列解码，保留CRF的标签依赖建模
