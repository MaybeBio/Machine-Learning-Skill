# Chapter 24: PageRank与潜在语义分析 / PageRank and Latent Semantic Analysis

> **导航**: 本章融合两个看似独立但共享矩阵分解思想的算法——PageRank（图排序）和潜在语义分析（文本降维），它们都利用矩阵特征值分解揭示数据中的隐含结构。

---

## 一、入门理解

### 1.1 PageRank：互联网的民主投票

想象一个学者在学术论文的引用网络中漫游：她随机打开一篇论文，从中随机选一条参考文献跳转过去，如此反复。经过足够长的时间，她停留在每篇论文上的时间比例就反映了该论文的重要性——被越多重要论文引用的论文，被访问的概率越高。

这就是PageRank的**随机游走模型**：
- 网页 = 节点，超链接 = 有向边
- 一个网页的PageRank值 = 随机游走者长期访问该网页的稳态概率
- 当一个网页被越多的"重要"网页链接，其PageRank值越高

**直观示例**：假设只有A、B、C三个网页。A链接到B和C，B链接到C，C链接到A。从任意页面出发随机点击链接，最终访问各页面的概率会收敛到某个稳态分布——这就是各页面的PageRank值。

### 1.2 潜在语义分析（LSA）：发现文本背后的"概念"

假设我们有数千篇文档，每篇文档都是一个长向量（词频统计）。在如此高维的空间中，文档之间的相似性难以直接计算。但一个关键洞察是：存在少数几个"潜在语义"（主题），使得文档空间可以被压缩到一个低维子空间中。

LSA的核心思想：
- **词-文档矩阵** $X$（$m \times n$，$m$ 个词，$n$ 篇文档）
- 每篇文档是词的线性组合，每个词是文档的线性组合
- 用**截断SVD**将 $X$ 分解为低秩近似，等价于找到词和文档在"主题空间"中的表示
- 在主题空间中用余弦相似度计算文档相似度，克服了同义词和多义词问题

**直观理解**：把"汽车"、"轿车"、"卡车"等词映射到同一个潜在主题维度上，把"银行"（金融机构）和"银行"（河岸）映射到不同的主题维度，从而在语义层面而非词汇层面理解文档。

### 1.3 两个算法的共同思想

PageRank和LSA看似不相关，但共享核心的数学工具：
- **PageRank**：网页链接矩阵的**主特征向量**（对应最大特征值的特征向量）
- **LSA**：词-文档矩阵的**截断SVD**（取前 $k$ 个最大奇异值对应的奇异向量）

二者都在矩阵分解的低维结构中揭示数据本质——网页的重要性排序和文档的语义结构。

## 二、公式推导

### 2.1 PageRank的随机游走模型推导

**基本模型**：定义网页转移矩阵 $M \in \mathbb{R}^{n \times n}$：

$$M_{ij} = \begin{cases} \frac{1}{d_j} & \text{如果页面 } j \text{ 有链接到页面 } i \\ 0 & \text{否则} \end{cases}$$

其中 $d_j$ 是页面 $j$ 的出链数（出度）。$M$ 的每一列之和为1（列随机矩阵），表示从页面 $j$ 出发等概率跳转到其链接的各页面。

**随机游走向量**：设 $r_t \in \mathbb{R}^n$ 为第 $t$ 步时在各页面的概率分布，则有：

$$r_{t+1} = M r_t$$

这是一个**马尔可夫链**。如果链是不可约（irreducible）且非周期（aperiodic）的，则存在唯一的稳态分布 $r^*$ 满足：

$$r^* = M r^*, \quad \sum_{i=1}^{n} r_i^* = 1$$

即 $r^*$ 是 $M$ 的特征值 $\lambda=1$ 对应的（归一化后的）特征向量。

### 2.2 阻尼因子与修正模型

**问题：Dead Ends和Spider Traps**

- **Dead Ends（悬挂节点）**：没有出链的页面，转移概率矩阵对应列全为0，不满足列随机性
- **Spider Traps（爬虫陷阱）**：一组网页只链接到组内的网页，导致随机游走者被困在其中

**Teleportation机制**：引入阻尼因子 $d$（通常取 $d = 0.85$）：

$$r_{t+1} = d \cdot M r_t + (1 - d) \cdot \frac{1}{n} \mathbf{1}$$

解释：随机游走者以概率 $d$ 按超链接前进，以概率 $1-d$ 随机跳转到任意页面。这确保了马尔可夫链的不可约性和非周期性，保证稳态分布存在且唯一。

**稳态方程**：

$$r^* = d M r^* + (1 - d) \frac{1}{n} \mathbf{1}$$

即 $r^* = (1 - d) (I - dM)^{-1} \frac{1}{n} \mathbf{1}$。

### 2.3 幂法（Power Method）求解PageRank

直接求 $(I - dM)^{-1}$ 对大规模矩阵不可行（网页数量达数十亿），常用**幂法迭代**求解：

**算法**：
1. 初始化 $r^{(0)} = \frac{1}{n} \mathbf{1}$
2. 重复直到收敛：$r^{(t+1)} = d M r^{(t)} + (1 - d) \frac{1}{n} \mathbf{1}$
3. $r^{(t+1)} = \frac{r^{(t+1)}}{\|r^{(t+1)}\|_1}$ （归一化）

在每次迭代后检查是否收敛（如 $\|r^{(t+1)} - r^{(t)}\|_1 < \epsilon$）。

**收敛性分析**：转移矩阵的谱半径由 $\lambda = d$ 的次大特征值控制（Perron-Frobenius定理保证最大特征值为1），收敛速率为 $O(d^t)$。当 $d=0.85$ 时，约50-100次迭代即可达到满意精度。

### 2.4 个性化与主题敏感PageRank

**个性化PageRank（Personalized PageRank）**：将随机跳转偏向一组特定的偏好页面 $\mathcal{P}$：

$$r_{t+1} = d M r_t + (1 - d) p$$

其中 $p$ 是偏好向量，在 $\mathcal{P}$ 中的页面上均匀分布，其余为0。

**主题敏感PageRank（Topic-Sensitive PageRank）**：预先计算不同主题（如体育、财经、科技等）的PageRank向量，查询时对用户的查询分类后使用对应主题的PageRank向量，或按主题相关性加权组合。

### 2.5 LSA的数学推导：截断SVD

**词-文档矩阵** $X \in \mathbb{R}^{m \times n}$，其中 $X_{ij}$ 是词 $i$ 在文档 $j$ 中的权重（通常为TF-IDF）：

$$\text{TF-IDF}(t, d) = \text{TF}(t, d) \cdot \log\frac{N}{\text{DF}(t)}$$

其中 $\text{TF}(t, d)$ 是词 $t$ 在文档 $d$ 中的词频，$N$ 是文档总数，$\text{DF}(t)$ 是包含词 $t$ 的文档数。

**SVD分解**：

$$X = U \Sigma V^{\mathrm{T}}$$

其中：
- $U \in \mathbb{R}^{m \times m}$：左奇异向量矩阵（词在主题空间的表示），$U^{\mathrm{T}}U = I$
- $\Sigma \in \mathbb{R}^{m \times n}$：对角矩阵，奇异值按递减排列 $\sigma_1 \geq \sigma_2 \geq \cdots \geq \sigma_r > 0$
- $V \in \mathbb{R}^{n \times n}$：右奇异向量矩阵（文档在主题空间的表示），$V^{\mathrm{T}}V = I$

**截断SVD**：只保留前 $k$ 个最大奇异值及其对应的奇异向量，得到 $X$ 的最优 $k$-秩近似：

$$X_k = U_k \Sigma_k V_k^{\mathrm{T}} = \arg\min_{\text{rank}(\hat{X}) \leq k} \|X - \hat{X}\|_F$$

**主题空间映射**：
- 词映射：$U_k \Sigma_k$ 或 $U_k$，每行是一个词在 $k$ 维主题空间中的坐标
- 文档映射：$\Sigma_k V_k^{\mathrm{T}}$ 或 $V_k$，每列是一篇文档在 $k$ 维主题空间中的坐标

**查询匹配**：查询 $q$（词频向量）映射到主题空间：$q_k = \Sigma_k^{-1} U_k^{\mathrm{T}} q$（或简化为 $U_k^{\mathrm{T}} q$），然后与文档向量计算余弦相似度：

$$\text{sim}(q, d_j) = \frac{q_k \cdot (V_k)_j}{\|q_k\| \|(V_k)_j\|}$$

### 2.6 截断SVD的最优性证明

Eckart-Young定理证明了截断SVD是Frobenius范数下的最优低秩近似：

$$\|X - X_k\|_F^2 = \sum_{i=k+1}^{r} \sigma_i^2$$

对于任意秩不超过 $k$ 的矩阵 $B$，有 $\|X - B\|_F \geq \|X - X_k\|_F$。

## 三、理论深化 —— 李航《机器学习方法》

### 3.1 PageRank的形式化体系（李航第2篇第21章）

李航将PageRank纳入**图排序**的统一框架，给出了系统化的形式定义。

**定义21.1（图排序模型）**：给定有向图 $G = (V, E)$，图排序的目标是计算每个节点的重要性得分 $s: V \mapsto \mathbb{R}^+$，使得重要性高的节点获得高分。

**PageRank的矩阵形式**：

$$s = d M s + \frac{1 - d}{n} \mathbf{1}$$

等价于求解线性方程组：

$$(I - dM) s = \frac{1 - d}{n} \mathbf{1}$$

**Perron-Frobenius定理**保证：当 $d \in (0, 1)$ 时，修正转移矩阵 $dM + \frac{1-d}{n} \mathbf{1} \mathbf{1}^{\mathrm{T}}$ 是正的列随机矩阵，存在唯一的主特征向量（对应特征值1），且所有分量均为正。

李航还讨论了PageRank的变体：
- 在有向图上的应用扩展到引用网络分析、社会网络影响力排序
- 调整阻尼因子 $d$ 可以控制全局流行度和局部链接结构的权重平衡

### 3.2 潜在语义分析的完整体系（李航第1篇第17章）

李航将LSA作为**话题分析（Topic Modeling）**的核心方法之一进行了详细阐述。

**定义17.1（词-文档矩阵）**：给定文档集合 $D = \{d_1, \ldots, d_n\}$ 和词汇表 $W = \{w_1, \ldots, w_m\}$，词-文档矩阵 $X = (x_{ij})_{m \times n}$，其中 $x_{ij}$ 是词 $w_i$ 在文档 $d_j$ 中的TF-IDF权值。

**定义17.2（话题向量空间）**：通过截断SVD获得的 $k$ 维空间称为**话题向量空间**，其中：
- 每个词 $w_i$ 对应于左奇异向量矩阵 $U_k$ 的第 $i$ 行向量
- 每个文档 $d_j$ 对应于右奇异向量矩阵 $V_k$ 的第 $j$ 行向量

**定义17.3（语义相似度）**：在话题向量空间中，任意两个对象的相似度通过其向量的内积或余弦值度量。

### 3.3 非负矩阵分解（NMF）与LSA的对比

李航还对NMF与LSA进行了深入对比：

| 特性 | LSA (SVD) | NMF |
|------|-----------|-----|
| 分解形式 | $X \approx U_k \Sigma_k V_k^{\mathrm{T}}$ | $X \approx WH$ |
| 约束 | 正交约束 $U^{\mathrm{T}}U=I, V^{\mathrm{T}}V=I$ | 非负约束 $W \geq 0, H \geq 0$ |
| 可解释性 | 奇异向量有正有负，难以直观解释 | 非负性使得每维直接对应可解释的聚类 |
| 解的唯一性 | 截断SVD唯一（奇异值不重） | 不唯一（非凸优化） |
| 计算复杂度 | $O(mn \cdot \min(m,n))$ | 迭代算法，依赖数据规模 |

**NMF的目标函数**：$\min_{W \geq 0, H \geq 0} \|X - WH\|_F^2$，用乘法更新规则迭代优化：

$$W_{ik} \leftarrow W_{ik} \frac{(XH^{\mathrm{T}})_{ik}}{(WHH^{\mathrm{T}})_{ik}}, \quad H_{kj} \leftarrow H_{kj} \frac{(W^{\mathrm{T}}X)_{kj}}{(W^{\mathrm{T}}WH)_{kj}}$$

### 3.4 从矩阵分解看两者的统一视角

PageRank和LSA共享的思想是：对数据矩阵执行特定的矩阵分解，在低维表示中揭示隐含结构。两者都处理了"大规模稀疏数据中的模式发现"问题，但干预对象不同——PageRank关注图结构中的节点排序，LSA关注文本集合中的语义结构。

## 四、综合总结

### 4.1 算法对比

| 维度 | PageRank | LSA |
|------|----------|-----|
| 输入 | 网页链接图 | 词-文档矩阵 |
| 核心思想 | 随机游走的稳态分布 | 截断SVD降维 |
| 数学工具 | 主特征向量 | 奇异值分解 |
| 输出 | 节点重要性排序 | 词与文档的主题坐标 |
| 目标 | 排序 | 语义表示与相似度计算 |
| 关键参数 | 阻尼因子 $d=0.85$ | 主题数 $k$ |
| 计算瓶颈 | 幂法迭代，$O(n^2)$ 稀疏 | SVD，$O(mn \min(m,n))$ |
| 扩展 | Personalized PageRank | pLSA, LDA |

### 4.2 关键结论

1. **PageRank的随机游走模型**将网页排序问题转化为马尔可夫链的稳态分布问题，阻尼因子的引入保证了收敛性和唯一性
2. **LSA通过截断SVD**在低维话题空间中实现"语义匹配"——即使两篇文档不共享词汇，在话题空间中也可有高相似度
3. **矩阵分解是二者的共同数学基础**：PageRank用主特征向量，LSA用截断SVD
4. **LSA的局限性**包括：缺乏概率解释（pLSA和LDA解决了此问题）、SVD计算开销大、主题数量 $k$ 需要人工设定
5. **PageRank的推广**：图中的各类随机游走模型（如HITS的Authority-Hub框架）在其他领域广泛使用
6. **NMF**作为LSA的有力替代，通过非负约束提供了更可解释的分解结果
