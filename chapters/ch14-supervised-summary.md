# 第14章: 监督学习方法总结 / Summary of Supervised Learning Methods

> **融合来源**: 李航第12章(监督学习方法总结) + 清华大学PPT + 各前章综合

## 一、入门理解

本章从统一视角梳理监督学习中各主要方法的共性、差异和适用场景，是前13章监督学习内容的"总复习"。

核心问题：给定训练集 $T=\{(\mathbf{x}_1,y_1),\ldots,(\mathbf{x}_m,y_m)\}$，在假设空间 $\mathcal{H}$ 中选择最优模型 $f$。所有方法都可归约为**模型+策略+算法**三要素的不同组合。

---

## 二、监督学习方法的统一分类

### 2.1 按模型类型

| 类型 | 代表方法 | 模型表示 | 特点 |
|------|---------|---------|------|
| **判别式-线性** | 感知机、LR、SVM(线性核) | $f(\mathbf{x})=\text{sign/convex}(\mathbf{w}^T\mathbf{x}+b)$ | 可解释、参数少 |
| **判别式-非线性** | SVM(核)、NN | $f(\mathbf{x})=\sum\alpha_i K(\mathbf{x}_i,\mathbf{x})$ 或深层复合 | 灵活、参数多 |
| **判别式-树结构** | 决策树 | 分区+常数/多数类 | 可解释、不稳定 |
| **生成式** | NB、LDA | $P(y)P(\mathbf{x}|y)$ | 可生成、可处理缺失 |
| **惰性学习** | k-NN | 存所有样本 | 无训练、预测慢 |

### 2.2 按学习策略

| 策略 | 数学形式 | 对应方法 |
|------|---------|---------|
| **ERM** | $\min_f \frac{1}{m}\sum_i L(y_i,f(\mathbf{x}_i))$ | 感知机(无正则化的特殊形式) |
| **SRM** | $\min_f \frac{1}{m}\sum_i L(y_i,f(\mathbf{x}_i)) + \lambda J(f)$ | SVM($\frac{1}{2}\|\mathbf{w}\|^2$)、LR(L2)、决策树(剪枝) |
| **MLE** | $\max_\theta \sum_i \log P(y_i|\mathbf{x}_i;\theta)$ | 逻辑回归(交叉熵)、朴素贝叶斯 |
| **MAP** | $\max_\theta \sum_i \log P(y_i|\mathbf{x}_i;\theta) + \log P(\theta)$ | 贝叶斯LR、NB(贝叶斯估计) |
| **最大间隔** | $\max \frac{2}{\|\mathbf{w}\|}$ s.t. constraints | SVM(硬/软间隔) |
| **前向分步+指数损失** | 逐轮 $\min_{(\alpha,G)}\sum_i e^{-y_i(f_{t-1}+\alpha G)}$ | AdaBoost |

### 2.3 损失函数一览

| 损失函数 | 公式 | 应用 | 性质 |
|---------|------|------|------|
| 0-1 | $\mathbb{I}(y \neq f(x))$ | 分类度量 | 非凸非光滑 |
| Hinge | $\max(0,1-yf(x))$ | SVM | 凸，0-1的代理 |
| 指数 | $e^{-yf(x)}$ | AdaBoost | 凸，0-1的代理 |
| 对数/交叉熵 | $\log(1+e^{-yf(x)})$ | LR | 凸，伯努利MLE |
| 平方 | $(y-f(x))^2$ | 回归 | 凸，高斯MLE |
| 绝对值 | $\|y-f(x)\|$ | 鲁棒回归 | 凸(非光滑)，拉普拉斯MLE |

## 三、各方法的联系与对比

### 3.1 线性分类器的统一视角

感知机 → 逻辑回归 → 线性SVM 的演变：

- **感知机**：$\min \sum_{mistake} -y_i(\mathbf{w}^T\mathbf{x}_i+b)$，无正则化，仅要求正确
- **逻辑回归**：$\min \sum \log(1+e^{-y_i(\mathbf{w}^T\mathbf{x}_i+b)}) + \lambda\|\mathbf{w}\|^2$，输出概率，加正则化
- **线性SVM**：$\min \sum \max(0, 1-y_i(\mathbf{w}^T\mathbf{x}_i+b)) + \lambda\|\mathbf{w}\|^2$，最大间隔

三者都寻找线性决策边界 $\mathbf{w}^T\mathbf{x}+b=0$，区别在于**损失函数**和**是否最大化间隔**。

### 3.2 决策树 vs 线性模型

| 维度 | 决策树 | 线性模型 |
|------|--------|---------|
| 决策边界 | 轴平行分段 | 线性超平面 |
| 特征尺度 | 不敏感 | 需标准化 |
| 特征交互 | 自动建模(树结构) | 需手动构造 |
| 可解释性 | 规则形式，很直观 | 权重排序，较直观 |
| 外推能力 | 无(叶结点常数) | 有(线性外推) |

### 3.3 生成式 vs 判别式

| 生成式 | 判别式 |
|--------|--------|
| $P(\mathbf{x},y) \to P(y|\mathbf{x})$ | 直接 $P(y|\mathbf{x})$ |
| 更"难"的估计 → 需更多假设 | 更"容易"的估计 → 假设更少 |
| 数据少时可能更好(利用了模型先验) | 数据多时通常更好 |
| 可生成数据、处理缺失值 | 仅能分类/回归 |
| NB, LDA, HMM | LR, SVM, CRF, NN |

## 四、综合总结

### 算法选择决策指南

| 场景 | 推荐算法 | 原因 |
|------|---------|------|
| 线性可分、可解释 | 感知机→LR | 简单、概率输出 |
| 高维文本分类 | 朴素贝叶斯、线性SVM | 稀疏特征效果好 |
| 非线性、中数据量 | RBF-SVM | 最大间隔+核技巧 |
| 非线性、大数据量 | GBDT、随机森林 | 快速、健壮 |
| 极度非线性、海量数据 | 深度神经网络 | 表示学习 |
| 需要概率输出 | LR、NB | 校准好的概率 |
| 类别不均衡 | 代价敏感SVM、集成 | 针对性优化 |
| 特征含缺失值 | 决策树、NB | 天然支持或简单填补 |

### 关键结论

1. **所有监督学习方法的本质**：在假设空间中极小化(结构)风险
2. **模型复杂度必须与数据量匹配**：$m$小→简单模型+强正则化；$m$大→可尝试复杂模型
3. **正则化 = 归纳偏置的数学实现**：L1→稀疏，L2→平滑，树剪枝→简短
4. **损失函数 = 数据生成假设的隐式编码**：平方损失→高斯噪声，交叉熵→伯努利/类别，Hinge→最大间隔
5. **没有任何算法在所有问题上都最好(NFL定理)**——选择必须匹配问题特点
