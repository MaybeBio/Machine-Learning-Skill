# Chapter 7: 贝叶斯分类器 / Bayesian Classifiers

## 核心概念 (Core Idea)

贝叶斯分类器基于概率框架进行决策：由贝叶斯定理从先验概率和类条件概率计算后验概率，选择使后验概率最大（或条件风险最小）的类别作为预测结果。其核心挑战在于如何从有限训练样本中准确估计类条件概率——朴素贝叶斯通过"属性条件独立"假设简化估计，贝叶斯网通过有向无环图显式建模属性依赖，而EM算法则为含隐变量情形提供了通用的参数估计框架。

---

## 融合框架 (Integrated Frameworks)

- **贝叶斯决策论**：最小化条件风险的贝叶斯判定准则；0/1损失下的最优分类器 (西瓜书)
- **极大似然估计**：频率学派视角下的参数估计方法，假设分布形式后最大化数据似然 (西瓜书)
- **朴素贝叶斯分类器**：属性条件独立性假设 + 概率估计 + 拉普拉斯平滑 (西瓜书/清华PPT)
- **半朴素贝叶斯**：放松独立性假设，ODE (SPODE, TAN, AODE) 允许每个属性依赖一个父属性 (西瓜书)
- **贝叶斯网**：有向无环图 + 条件概率表的完整表示；d-分离与条件独立性；评分搜索结构学习；吉布斯采样近似推断 (西瓜书/清华PPT)
- **EM算法**：期望-最大化交替迭代，解决含隐变量的参数估计问题 (西瓜书)

---

## 理论推导 (Theoretical Derivation)

### 1. 贝叶斯决策论

设 $N$ 种类别标记 $\mathcal{Y} = \{c_1, \ldots, c_N\}$，$\lambda_{ij}$ 为将 $c_j$ 误分类为 $c_i$ 的损失。样本 $\mathbf{x}$ 上的条件风险为：

$$R(c_i \mid \mathbf{x}) = \sum_{j=1}^{N} \lambda_{ij} P(c_j \mid \mathbf{x})$$

贝叶斯判定准则：$h^*(\mathbf{x}) = \arg\min_{c \in \mathcal{Y}} R(c \mid \mathbf{x})$

对于0/1损失（最小化错误率），最优分类器简化为：

$$h^*(\mathbf{x}) = \arg\max_{c \in \mathcal{Y}} P(c \mid \mathbf{x})$$

由贝叶斯定理：

$$P(c \mid \mathbf{x}) = \frac{P(c)P(\mathbf{x} \mid c)}{P(\mathbf{x})}$$

其中 $P(c)$ 为先验概率，$P(\mathbf{x} \mid c)$ 为似然（类条件概率），$P(\mathbf{x})$ 为归一化因子。

### 2. 极大似然估计 (MLE)

假设类条件概率 $P(\mathbf{x} \mid c)$ 具有参数形式 $P(\mathbf{x} \mid \boldsymbol{\theta}_c)$，其中 $\boldsymbol{\theta}_c$ 为参数向量。

对数似然：$LL(\boldsymbol{\theta}_c) = \sum_{\mathbf{x} \in D_c} \log P(\mathbf{x} \mid \boldsymbol{\theta}_c)$

极大似然估计：$\hat{\boldsymbol{\theta}}_c = \arg\max_{\boldsymbol{\theta}_c} LL(\boldsymbol{\theta}_c)$

例如，假设 $p(\mathbf{x} \mid c) \sim \mathcal{N}(\boldsymbol{\mu}_c, \boldsymbol{\sigma}_c^2)$，则MLE给出 $\hat{\boldsymbol{\mu}}_c = \frac{1}{|D_c|}\sum_{\mathbf{x} \in D_c} \mathbf{x}$，$\hat{\boldsymbol{\sigma}}_c^2 = \frac{1}{|D_c|}\sum_{\mathbf{x} \in D_c} (\mathbf{x} - \hat{\boldsymbol{\mu}}_c)(\mathbf{x} - \hat{\boldsymbol{\mu}}_c)^\top$，即样本均值和样本协方差。

### 3. 朴素贝叶斯 (Naive Bayes)

**核心假设**：属性条件独立性——给定类别时，各属性相互独立。

$$P(c \mid \mathbf{x}) \propto P(c) \prod_{i=1}^{d} P(x_i \mid c)$$

**参数估计**（频率学派）：

- 先验概率：$P(c) = \frac{|D_c|}{|D|}$
- 离散属性：$P(x_i \mid c) = \frac{|D_{c, x_i}|}{|D_c|}$
- 连续属性（假设正态分布）：$p(x_i \mid c) = \frac{1}{\sqrt{2\pi}\sigma_{c,i}} \exp\left(-\frac{(x_i - \mu_{c,i})^2}{2\sigma_{c,i}^2}\right)$

**拉普拉斯平滑**：为避免训练集中未出现的属性值导致概率为零：

$$\hat{P}(c) = \frac{|D_c| + 1}{|D| + N}, \quad \hat{P}(x_i \mid c) = \frac{|D_{c, x_i}| + 1}{|D_c| + N_i}$$

其中 $N$ 为类别数，$N_i$ 为第 $i$ 个属性的取值数。

### 4. 半朴素贝叶斯 (Semi-Naive Bayes)

ODE (One-Dependent Estimator) 允许每个属性最多依赖一个父属性 $pa_i$：

$$P(c \mid \mathbf{x}) \propto P(c) \prod_{i=1}^{d} P(x_i \mid c, pa_i)$$

- **SPODE**：所有属性共享同一个超父属性，通过交叉验证选择
- **TAN**：基于条件互信息 $I(x_i, x_j \mid c)$ 构建最大带权生成树，确定父子关系
- **AODE**：将每个属性作为超父构建SPODE后集成，需要 $|D_{x_i}| \geq m'$ 的数据支撑

### 5. 贝叶斯网 (Bayesian Network)

贝叶斯网 $B = \langle G, \Theta \rangle$ 用有向无环图 $G$ 编码属性间依赖关系，用条件概率表 $\Theta$ 量化依赖强度。

联合概率分解：

$$P_B(x_1, \ldots, x_d) = \prod_{i=1}^{d} P_B(x_i \mid \pi_i) = \prod_{i=1}^{d} \theta_{x_i | \pi_i}$$

其中 $\pi_i$ 为 $x_i$ 的父结点集。

**三种基本结构**：

- **同父结构**：$x_1 \to x_3, x_4$，给定 $x_1$ 时 $x_3 \perp x_4$
- **顺序结构**：$x \to y \to z$，给定 $y$ 时 $x \perp z$
- **V型结构（冲撞结构）**：$x_1, x_2 \to x_4$，$x_4$ 未知时 $x_1 \perp x_2$；$x_4$ 已知时 $x_1$ 与 $x_2$ 必不独立

**d-分离**：通过道德图（Moral Graph）判断条件独立性——将V型结构的父结点之间加边、所有有向边变无向边后，若去除变量集 $\mathbf{z}$ 后 $x$ 和 $y$ 处于不同连通分量，则 $x \perp y \mid \mathbf{z}$。

**结构学习**：评分函数 $s(B \mid D) = f(\theta)|B| - LL(B \mid D)$，常用BIC准则 $BIC(B \mid D) = \frac{\log m}{2}|B| - LL(B \mid D)$。在固定结构下参数可由频率直接估计；结构搜索是NP难问题，通常用贪心法或限定为树形结构。

**近似推断（吉布斯采样）**：将证据变量固定，对查询变量按其条件分布随机采样更新，利用马尔可夫链的平稳分布估计后验概率。

### 6. EM算法

当存在隐变量 $\mathbf{Z}$ 时，对数边际似然为：

$$LL(\Theta \mid \mathbf{X}) = \ln \sum_{\mathbf{Z}} P(\mathbf{X}, \mathbf{Z} \mid \Theta)$$

EM算法交替执行：

- **E步**：以当前参数 $\Theta^t$ 推断隐变量分布，计算期望似然 $Q(\Theta \mid \Theta^t) = \mathbb{E}_{\mathbf{Z} \mid \mathbf{X}, \Theta^t}[\ln P(\mathbf{X}, \mathbf{Z} \mid \Theta)]$
- **M步**：最大化期望似然 $\Theta^{t+1} = \arg\max_{\Theta} Q(\Theta \mid \Theta^t)$

EM算法收敛到对数似然的局部最优解，可视为通过坐标下降法最大化似然下界。

## 核心概念 (Key Concepts)

- **先验概率 $P(c)$ vs 后验概率 $P(c \mid \mathbf{x})$**：先验是看到数据前对类别的信念，后验是综合证据后的修正信念
- **类条件概率 $P(\mathbf{x} \mid c)$**：在类别 $c$ 下观测到 $\mathbf{x}$ 的概率（似然），是生成式模型的核心
- **判别式 vs 生成式**：判别式直接建模 $P(c \mid \mathbf{x})$；生成式建模 $P(\mathbf{x}, c) = P(c)P(\mathbf{x} \mid c)$
- **贝叶斯最优分类器**：达到贝叶斯风险（理论上限）的分类器——在各模型假设下是"天花板"
- **属性条件独立性**：朴素贝叶斯的核心假设——实践中常不成立，但排序结果往往仍然正确
- **拉普拉斯平滑**：在分子加1、分母加取值数，等价于引入均匀先验以解决零概率问题
- **条件互信息**：$I(x_i, x_j \mid y)$ 量化已知类别下两属性的相关性，是TAN构建树结构的基础
- **吉布斯采样**：马尔可夫链蒙特卡洛（MCMC）方法，在贝叶斯网推断中逐变量采样以近似后验分布

## 算法步骤 (Algorithm Steps)

### 朴素贝叶斯训练与预测

1. **训练阶段**：
   - 对每个类别 $c$，计算先验概率 $\hat{P}(c) = (|D_c| + 1) / (|D| + N)$
   - 对每个属性 $x_i$ 和类别 $c$：
     - 离散属性：计算条件概率 $\hat{P}(x_i \mid c) = (|D_{c, x_i}| + 1) / (|D_c| + N_i)$
     - 连续属性：估计 $\mu_{c,i}$ 和 $\sigma_{c,i}^2$，存储分布参数
2. **预测阶段**：
   - 对测试样本 $\mathbf{x}$：
     $$c^* = \arg\max_{c} \left[ \log \hat{P}(c) + \sum_{i=1}^{d} \log \hat{P}(x_i \mid c) \right]$$
     （取对数防止连乘下溢）

### TAN构建

1. 计算每对属性间的条件互信息 $I(x_i, x_j \mid y)$
2. 以属性为结点、$I(x_i, x_j \mid y)$ 为边权重构建完全图
3. 运行最大带权生成树算法，选根结点并定向边
4. 加入类别结点，增加从 $y$ 到每个属性的有向边

### EM算法 (西瓜书标准版)

1. 初始化参数 $\Theta^0$
2. **Repeat**：
   - E步：基于 $\Theta^t$ 推断隐变量分布，计算 $Q(\Theta \mid \Theta^t)$
   - M步：$\Theta^{t+1} = \arg\max_{\Theta} Q(\Theta \mid \Theta^t)$
3. **Until** 参数变化小于阈值或对数似然收敛
4. 输出：$\Theta^*$

## 直观理解 (Intuition)

**朴素贝叶斯如"拆零为整的侦探"**：一名侦探调查一个案件（分类），他不会试图一次性理解所有线索（属性）的联动关系，而是假设每条线索独立地指向嫌疑人。尽管这个假设通常不完全正确，但在大多数案件中，综合每条线索的独立判断已经足以指认真凶。

**MLE如"眼见为实"**：假设你观测到一枚硬币10次抛出7次正面，你会自然地估计正面概率为0.7——因为在你看到的10次里7次是正面。这就是极大似然——用"看到的事实"来推测"背后的参数"。

**拉普拉斯平滑如"留条后路"**：如果一个属性值从未在训练数据中出现，直接说"概率为零"就太武断了。拉普拉斯平滑相当于说"我姑且假设每个值都至少出现了一次"——这是一种谨慎的乐观。

**贝叶斯网如"家族关系谱"**：属性之间的依赖关系像是一个家族谱。有的属性是"家长"（父结点，如"好瓜"决定"色泽"），有的是"孩子"，而"冲撞结构"揭示了最反直觉的现象——两个独立的原因，在观察到共同结果后反而变得不再独立（比如发烧和中毒，在已知你进了医院后变得相互关联）。

**EM算法如"盲人摸象"**：如果你被蒙上眼睛，要估计房间里的家具布局——你只能先猜测布局（E步：假设参数已知来推断隐变量），然后伸手去摸验证（M步：根据触感修正参数）。反复交替，越猜越准。

## 常见误区 (Anti-patterns)

- **误区："朴素贝叶斯的独立性假设太强，所以性能差"**：事实上朴素贝叶斯在很多任务（尤其是文本分类）中表现优异。关键在于分类只需概率排序正确，无需精确概率值；当依赖关系对所有类别影响一致或相互抵消时，独立性假设不会损害分类结果
- **误区："生成式模型总是优于判别式模型"**：Asymptotically, 生成式模型确实能以更少样本达到更好性能。但在有限样本下，如果模型假设与真实分布差距大，判别式模型（如SVM、逻辑回归）往往更鲁棒
- **误区："贝叶斯网结构学习可以通过贪心法找到全局最优"**：结构学习是NP难问题。贪心法只能保证局部最优，实际上严重依赖初始结构和搜索策略
- **误区："EM算法保证收敛到全局最优"**：EM算法只能保证收敛到对数似然的局部极值点。实际使用时需要多组随机初始化或配合其他策略（如模拟退火）来避免陷入差的局部最优
- **误区："零概率问题只影响朴素贝叶斯"**：所有基于频率统计的模型都会遇到。在贝叶斯网中，条件概率表同样需要平滑，否则吉布斯采样可能因零概率而给出错误估计

## 代码要点 (Code Essentials)

- **数值稳定性**：用对数空间计算避免下溢：$\log P(c) + \sum_i \log P(x_i \mid c)$ 替代 $P(c)\prod_i P(x_i \mid c)$
- **连续属性处理**：朴素贝叶斯对连续属性通常假设高斯分布（GaussianNB），要求大致满足正态分布假设；若不满足，可考虑核密度估计(KDE)或分箱离散化
- **增量学习**：朴素贝叶斯的参数估计都是基于计数，天然支持增量学习——新样本到达时只需更新计数器即可
- **缺失值处理**：计算概率时忽略缺失的属性维度（即只考虑观测到的属性进行乘积）
- **AODE实现注意**：需要过滤掉训练样本数不足的属性值（阈值 $m'$ 默认为30）
- **贝叶斯网结构学习**：pgmpy 等库提供了Hill Climbing、PC等算法，BIC评分比AIC评分对大样本有更严格的正则化
- **EM初始化**：多组随机初始化并选择最高对数似然的结果；GMM是EM算法的典型应用案例（参见聚类章节）

## 实例分析 (Worked Example)

**西瓜数据集3.0上的朴素贝叶斯分类** (西瓜书/清华PPT):

训练集共17个样本，8好瓜、9坏瓜。对测试样例 "测1"（色泽=青绿、根蒂=蜷缩、敲声=浊响、纹理=清晰、脐部=凹陷、触感=硬滑、密度=0.697、含糖率=0.460）：

先验概率：$P(\text{好瓜}) = 8/17 \approx 0.471$，$P(\text{坏瓜}) = 9/17 \approx 0.529$

离散属性频率估计（仅列两项示例）：

- $P(\text{色泽=青绿} \mid \text{好瓜}) = 3/8 = 0.375$；$P(\text{色泽=青绿} \mid \text{坏瓜}) = 3/9 \approx 0.333$

连续属性高斯估计：

- 好瓜：密度均值0.574，标准差0.129 → $p(0.697 \mid \text{好瓜}) \approx 1.959$
- 坏瓜：密度均值0.496，标准差0.195 → $p(0.697 \mid \text{坏瓜}) \approx 1.203$

连乘比较（取对数）：好瓜的乘积约为0.038，坏瓜的乘积约为 $6.80 \times 10^{-5}$。由于 $0.038 > 6.80 \times 10^{-5}$，分类为"好瓜"。

若对"敲声=清脆"的样本（训练集中好瓜无此值），直接估计会得零概率。拉普拉斯修正后：$\hat{P}(\text{敲声=清脆} \mid \text{好瓜}) = (0+1)/(8+3) \approx 0.091$，避免了零概率问题。

## 关键结论 (Key Takeaways)

1. 贝叶斯决策论给出了概率框架下的最优决策策略——但前提是精确知道后验概率，这在现实中几乎不可能
2. 朴素贝叶斯以其极简的结构和"令人惊讶的好性能"成为数据挖掘十大算法之一，尤其适用于文本分类
3. 拉普拉斯平滑是频率估计中最简单有效的修正方法，实质上是引入均匀先验
4. 贝叶斯网提供了从"完全独立"(NB)到"完全依赖"(全贝叶斯网)之间的灵活谱系，但结构学习是计算瓶颈
5. EM算法是处理隐变量的标准工具，其E步推断+M步最大化的框架在机器学习中应用极为广泛（GMM、HMM、缺失数据等）
6. 生成式模型与判别式模型各有优劣，选择取决于数据规模和模型假设是否符合真实分布
