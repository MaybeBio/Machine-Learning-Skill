## 一、入门理解 — 周志华《机器学习》(西瓜书)

### 1.1 无监督学习的任务全景

无监督学习从无标注数据中学习数据的统计规律或内在结构。根据训练数据是否拥有标记信息，机器学习被划分为监督学习和无监督学习两大类。聚类是无监督学习的代表，但还包括：

- **密度估计**(density estimation)：估计数据的概率分布
- **异常检测**(anomaly detection)：发现与众不同的样本
- **降维**(dimensionality reduction)：从高维到低维
- **话题分析**(topic analysis)：从文本中自动发现主题
- **图分析**(graph analysis)：分析图结构数据(如PageRank)

### 1.2 无监督学习的基本想法

西瓜书将无监督学习目标概括为：通过对无标记训练样本的学习揭示数据的内在性质和规律。其基本想法是对给定数据进行某种"压缩"，从而发掘数据的潜在结构：

- **纵向结构**（聚类）：把相似的样本聚到同类
- **横向结构**（降维）：把高维向量转换为低维向量
- **纵横向结构**（话题模型/概率图模型）：假设数据由含隐结构的概率模型生成

## 二、公式推导 — 南瓜书补充

### 2.1 无监督学习的统一损失函数视角

南瓜书将无监督学习中的模型统一为某种重构损失最小化：

- k-means：$\min_{\{\boldsymbol{\mu}_i\},\{r_{ij}\}} \sum_{i=1}^{k}\sum_{j=1}^{m} r_{ij}\|\mathbf{x}_j - \boldsymbol{\mu}_i\|^2$，其中 $r_{ij} \in \{0,1\}$ 为簇分配指示
- PCA：$\min_{\mathbf{W}} \sum_{i=1}^{m}\|\mathbf{x}_i - \mathbf{W}\mathbf{W}^T\mathbf{x}_i\|^2$（重构误差）
- LSA：$\min_{\mathbf{U},\mathbf{V}} \|\mathbf{X} - \mathbf{U}\boldsymbol{\Sigma}\mathbf{V}^T\|_F^2$（矩阵分解误差）
- Autoencoder：$\min_{\phi,\psi} \sum_i \|\mathbf{x}_i - \psi(\phi(\mathbf{x}_i))\|^2$

它们都试图以某种简洁表示来"解释"原始数据。

### 2.2 NMF(非负矩阵分解)的推导

南瓜书给出NMF的乘法更新规则推导。对非负矩阵 $\mathbf{X} \in \mathbb{R}^{m \times n}$，分解为 $\mathbf{X} \approx \mathbf{W}\mathbf{H}$ 其中 $\mathbf{W} \geq 0, \mathbf{H} \geq 0$。目标函数为Frobenius范数最小化：

$$J = \frac{1}{2}\|\mathbf{X} - \mathbf{W}\mathbf{H}\|_F^2$$

梯度下降得到乘法更新规则：
$$H_{kj} \leftarrow H_{kj}\frac{(\mathbf{W}^T\mathbf{X})_{kj}}{(\mathbf{W}^T\mathbf{W}\mathbf{H})_{kj}}, \quad W_{ik} \leftarrow W_{ik}\frac{(\mathbf{X}\mathbf{H}^T)_{ik}}{(\mathbf{W}\mathbf{H}\mathbf{H}^T)_{ik}}$$

## 三、理论深化 — 李航《机器学习方法》+ 清华大学PPT

### 3.1 无监督学习三要素统一表（李航第22章核心）

李航第22章将无监督学习方法以三要素框架统一总结：

| 场景 | 方法 | 模型 | 策略 | 算法 |
|------|------|------|------|------|
| **聚类** | 层次聚类 | 聚类树 | 类内距离最小 | 启发式算法 |
| | k均值聚类 | k个中心 | 样本与中心距离最小 | 迭代算法 |
| | GMM | 高斯混合模型 | 似然函数最大 | EM算法 |
| **降维** | PCA | 低维正交空间 | 方差最大 | SVD |
| **话题分析** | LSA | 矩阵分解模型 | 平方损失最小 | SVD |
| | NMF | 矩阵分解模型 | 平方损失最小 | 乘法更新 |
| | PLSA | 概率生成模型 | 似然函数最大 | EM算法 |
| | LDA | 贝叶斯概率模型 | 后验概率估计 | Gibbs/变分 |
| **图分析** | PageRank | 有向图马尔可夫链 | 平稳分布 | 幂法 |

### 3.2 话题模型的统一视角

李航特别强调四个话题模型的统一：LSA/NMF/PLSA/LDA可在概率图模型框架下统一理解。

**概率解释**：
- LSA和NMF：$d_n \sim \mathcal{N}(\mathbf{U}\mathbf{v}_n, \sigma^2\mathbf{I})$（高斯分布）
- PLSA：多项分布，不含先验
- LDA：多项分布 + Dirichlet先验（贝叶斯化）

四个模型间的关系：
- LSA → PLSA：将确定性SVD替换为概率生成模型
- PLSA → LDA：为参数添加Dirichlet先验，实现贝叶斯推断
- NMF ↔ LSA：NMF可视为LSA的非负约束版本

### 3.3 PPT补充：无监督学习方法的共性与差异

PPT总结了无监督方法的三个共性：
1. **都依赖于数据自身结构**（无外部监督信号）
2. **都需要定义"好"的标准**（紧密度/方差/似然/平稳分布）
3. **都面临模型选择问题**（聚类数k/降维维数/话题数）

关键差异在于**优化问题的性质**：
- 凸优化：PCA（全局最优保证）
- 非凸优化：k-means、GMM（收敛到局部最优）
- 组合优化：层次聚类（启发式求解）

## 四、综合总结

### 方法全景对比

| 任务 | 代表方法 | 核心技术 | 输出形式 | 是否概率 |
|------|---------|---------|---------|---------|
| 聚类 | k-means | 迭代优化 | 簇分配 | 否 |
| 聚类 | GMM | EM算法 | 软分配+概率 | 是 |
| 降维 | PCA | SVD | 低维坐标 | 否 |
| 降维 | Autoencoder | BP算法 | 低维编码 | 否 |
| 话题 | LSA | SVD | 词-话题矩阵 | 否 |
| 话题 | PLSA | EM算法 | 概率分布 | 是 |
| 话题 | LDA | Gibbs/VI | 后验分布 | 是 |
| 图分析 | PageRank | 幂法 | 结点重要度 | 概率解释 |

### 关键结论

1. **无监督学习发掘数据的三种结构**：纵向(聚类)、横向(降维)、纵横(话题/图模型)
2. **三要素框架同样适用无监督学习**：模型(簇结构/子空间/概率分布)、策略(误差最小/方差最大/似然最大)、算法(迭代/EM/SVD/采样)
3. **概率化是重要趋势**：从确定性分解(LSA)→概率模型(PLSA)→贝叶斯模型(LDA)
4. **SVD是无监督学习的基础工具**：直接用于LSA、PCA，间接通过矩阵分解连接多种方法
5. **EM算法是含隐变量无监督学习的核心算法**：统一了GMM、PLSA等方法的求解
6. **无监督评估更具挑战性**：缺乏ground truth，依赖内部或外部有效性指标
7. **无监督学习常作为监督学习的前处理**：聚类→分类标签、降维→特征预处理
